# 退换货政策

## 退货时效

- 签收之日起 **7日内** 可申请无理由退货（定制款除外）
- 收到退回商品之日起 **7日内** 退款或换货

## 退货条件

| 条件 | 说明 |
|------|------|
| 商品完好 | 吊牌未剪、未使用、未洗涤、无穿着痕迹 |
| 包装完好 | 鞋盒无破损、涂画、缠绕胶带，吊牌完整 |
| 附件齐全 | 购物凭证、发票、赠品、装箱清单一并退回 |
| 包装方式 | 不能用商品鞋盒/服装袋直接寄回，外面必须加快递纸盒 |

## 不予退货的情况

- 商品使用不当造成的受损、褪色、脱线、开裂
- 赠品
- 非本官方商城发出的商品
- 无购买记录的商品
- 恶意退货

## 运费

- **质量问题**：Onitsuka Tiger承担运费（联系客服确认推荐快递，邮费截图报销）
- **非质量问题**：买家承担退回运费

## 退款方式

- 原路退回（微信/支付宝/花呗分期）

## 物流

- 需使用快递送货上门，不接受收件方上门取货的物流方式
